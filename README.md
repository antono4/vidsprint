# Vidsprint 🎬

![Vidsprint Banner](https://images.unsplash.com/photo-1574717024653-61fd2cf4d44d?w=800)

Video creation and editing tool for quick content production.

## ✨ Features

- 🎞️ Video trimming
- 🎨 Filters and effects
- 🎵 Audio editing
- 📝 Text overlay
- 💾 Export in multiple formats

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Canvas API

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/antono4/vidsprint.git

# Open in browser
cd vidsprint
open index.html
```

## 📁 Project Structure

```
vidsprint/
├── index.html      # Main page
├── css/            # Stylesheets
├── js/             # JavaScript files
└── assets/         # Static assets
```
